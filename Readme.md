# Task Tracker Application

## Technology Used
* **Backend Framework:** .NET 10.0 (For api)
* **Programming Language:** C#
* **Frontend Languages:** HTML, CSS, Javascript (for communicating with api from backend)
* **Data Storage:** JSON file (`tasks.json`)

## Features Completed
- Add task
- View task list
- Mark task as completed
- Delete task
- Timestamp of task

## How to Run

Follow these step to build and run this .NET project (Task Tracker)

### 1. Prerequisite Check
Install .NET sdk for windows or Ubuntu from official sources

### 2. Open this project directory

### 3. Build the project from root directory of the project
```bash
dotnet build
```

### 4. Run the .NET project
```bash
dotnet run
```
Open browser and go to the url: `http://localhost:5119` where application is running
